# DasBuch
C Pointer and Memory – Study and Implementation

This repository contains a structured study of pointer and memory concepts in C, based on:
“C – The Pointer Book” by Klaus Schröder
The work combines theoretical study, practical implementation, and structured documentation.

Scope
The repository includes:
Chapter-based solutions to pointer and memory exercises
C implementations focusing on low-level behavior
A structured written document (thesis-style, Word)
The original reference book (PDF)

Approach
The study follows a systematic workflow:
Understanding memory representation and addressing
Implementing exercises in C
Analyzing pointer behavior and memory effects
Documenting concepts in a structured, academic format

Topics
Pointer arithmetic and dereferencing
Memory layout (stack / heap)
Arrays and multidimensional structures
Structures and alignment
Dynamic memory management

The goal is to develop a clear understanding of how memory works in C and how pointer operations affect program behavior at a low level.
